Difference between Session storage and Local storage

Local storage: Local storage stores a larger amount of data on the client's computer in a key-value pair format.Local storage data has no expiration date.Data is never transferred to the server and is accessible from any window.Local storage limit about 5MB per app per browser. localStorage data are shared between all tabs and windows from the same origin.

Session storage: Session storage stores a larger amount of data on the client's computer in a key-value pair format.Session storage only for the current session, expiring the data on tab close.Data is never transferred to the server and is accessible from the same tab.Session storage limit about 5MB. If we open multiple tabs/windows with the same URL it creates a sessionStorage for each tab/window.
